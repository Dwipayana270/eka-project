Faktor dan Pembuat Nol Polinomial serta Identitas Polinomial

Memahami konsep faktor, akar, dan identitas polinomial

Pembuat Nol (Akar) Polinomial

Pembuat nol (akar) polinomial adalah nilai x yang membuat P(x) = 0. Jika P(k) = 0, maka k adalah akar dari polinomial P(x).

Contoh: P(x) = x² - 5x + 6

P(2) = 4 - 10 + 6 = 0 → x = 2 adalah akar

P(3) = 9 - 15 + 6 = 0 → x = 3 adalah akar

Faktor Polinomial

Jika (x - k) adalah faktor dari P(x), maka k adalah akar dari P(x) (P(k) = 0).

Teorema Faktor: (x - k) adalah faktor dari P(x) jika dan hanya jika P(k) = 0

Menentukan Faktor-Faktor Polinomial

1. Metode Faktorisasi

Mencari faktor dengan memfaktorkan polinomial.

Contoh: Faktorkan P(x) = x² - 5x + 6

P(x) = (x - 2)(x - 3)

Faktor: (x - 2) dan (x - 3)

Akar: x = 2 dan x = 3

2. Menggunakan Teorema Faktor

Mencari nilai k yang membuat P(k) = 0, kemudian menggunakan pembagian polinomial.

Contoh: Faktorkan P(x) = x³ - 6x² + 11x - 6

Coba x = 1: P(1) = 1 - 6 + 11 - 6 = 0 → (x - 1) adalah faktor

Bagi P(x) dengan (x - 1): hasilnya x² - 5x + 6

Faktorkan: x² - 5x + 6 = (x - 2)(x - 3)

Jadi, P(x) = (x - 1)(x - 2)(x - 3)

Jumlah dan Hasil Kali Akar-Akar Polinomial

Untuk Polinomial Kuadrat: ax² + bx + c = 0

Jika akar-akarnya x₁ dan x₂, maka:

• x₁ + x₂ = -b/a

• x₁ × x₂ = c/a

Untuk Polinomial Kubik: ax³ + bx² + cx + d = 0

Jika akar-akarnya x₁, x₂, dan x₃, maka:

• x₁ + x₂ + x₃ = -b/a

• x₁x₂ + x₁x₃ + x₂x₃ = c/a

• x₁x₂x₃ = -d/a

Untuk Polinomial Derajat n: aₙxⁿ + aₙ₋₁xⁿ⁻¹ + ... + a₀ = 0

• Jumlah akar-akar = -aₙ₋₁/aₙ

• Jumlah hasil kali akar-akar dua-dua = aₙ₋₂/aₙ

• Jumlah hasil kali akar-akar tiga-tiga = -aₙ₋₃/aₙ

• Hasil kali semua akar = (-1)ⁿ × a₀/aₙ

Identitas Polinomial

Dua polinomial dikatakan identik jika mempunyai derajat yang sama dan koefisien-koefisien yang bersesuaian sama.

Contoh: Tentukan nilai a, b, dan c jika:

2x² + 3x - 1 ≡ a(x - 1)² + b(x - 1) + c

Penyelesaian:

Ruas kanan: a(x² - 2x + 1) + b(x - 1) + c = ax² + (-2a + b)x + (a - b + c)

Bandingkan dengan ruas kiri:

a = 2

-2a + b = 3 → -4 + b = 3 → b = 7

a - b + c = -1 → 2 - 7 + c = -1 → c = 4

Menentukan Polinomial dari Akar-Akarnya

Jika diketahui akar-akar polinomial, dapat dibentuk polinomialnya menggunakan rumus:

P(x) = a(x - x₁)(x - x₂)...(x - xₙ)

Contoh: Tentukan polinomial derajat 3 dengan akar-akar 1, 2, dan -3

P(x) = a(x - 1)(x - 2)(x + 3)

Jika a = 1, maka P(x) = (x - 1)(x - 2)(x + 3)

P(x) = (x² - 3x + 2)(x + 3) = x³ - 3x² + 2x + 3x² - 9x + 6 = x³ - 7x + 6

Latihan Soal

1. Tentukan akar-akar dari P(x) = x³ - 3x² - 4x + 12

2. Jika akar-akar persamaan 2x³ - 5x² + 4x - 1 = 0 adalah α, β, γ, hitung:

a. α + β + γ

b. αβ + αγ + βγ

c. αβγ

3. Tentukan nilai a dan b jika (x - 1) dan (x + 2) adalah faktor dari P(x) = x³ + ax² + bx - 6

4. Buktikan bahwa x² + y² + z² ≡ (x + y + z)² - 2(xy + xz + yz)
