Pengertian Polinomial

Polinomial (suku banyak) adalah ekspresi aljabar yang terdiri dari variabel dan koefisien, yang hanya melibatkan operasi penjumlahan, pengurangan, perkalian, dan pangkat bilangan bulat non-negatif.

Bentuk Umum Polinomial:

P(x) = aₙxⁿ + aₙ₋₁xⁿ⁻¹ + ... + a₂x² + a₁x + a₀

Dimana:

  • aₙ, aₙ₋₁, ..., a₀ adalah koefisien (aₙ ≠ 0)
  • n adalah derajat polinomial (bilangan bulat non-negatif)
  • x adalah variabel
  • a₀ adalah konstanta

Komponen Polinomial

1. Suku: Setiap bagian yang dipisahkan oleh tanda + atau -

2. Koefisien: Angka yang mengalikan variabel

3. Derajat: Pangkat tertinggi dari variabel

4. Konstanta: Suku tanpa variabel

Contoh Polinomial

1. P(x) = 3x⁴ - 2x³ + 5x - 7

  • Derajat: 4
  • Koefisien: 3, -2, 0, 5, -7
  • Konstanta: -7

2. Q(x) = 2x³ + 4x² - x + 8

  • Derajat: 3
  • Koefisien: 2, 4, -1, 8
  • Konstanta: 8

Fungsi Polinomial

Fungsi polinomial adalah fungsi yang dinyatakan dalam bentuk polinomial. Setiap nilai x yang dimasukkan akan menghasilkan nilai P(x) tertentu.

Contoh: Jika P(x) = 2x² - 3x + 1

Maka P(2) = 2(2)² - 3(2) + 1 = 8 - 6 + 1 = 3

Jenis-Jenis Polinomial Berdasarkan Derajat

1. Polinomial Konstan: Derajat 0 (contoh: P(x) = 5)

2. Polinomial Linear: Derajat 1 (contoh: P(x) = 2x + 3)

3. Polinomial Kuadrat: Derajat 2 (contoh: P(x) = x² - 4x + 4)

4. Polinomial Kubik: Derajat 3 (contoh: P(x) = x³ - 2x² + x - 1)

5. Polinomial Kuartik: Derajat 4 (contoh: P(x) = x⁴ + 2x² - 3)

Operasi Dasar Polinomial

1. Penjumlahan: Menjumlahkan koefisien suku-suku sejenis

Contoh: (3x² + 2x - 1) + (x² - 3x + 4) = 4x² - x + 3

2. Pengurangan: Mengurangkan koefisien suku-suku sejenis

Contoh: (3x² + 2x - 1) - (x² - 3x + 4) = 2x² + 5x - 5

3. Perkalian: Mengalikan setiap suku dengan setiap suku lainnya

Contoh: (x + 2)(x - 3) = x² - 3x + 2x - 6 = x² - x - 6

Latihan Soal

1. Tentukan derajat, koefisien, dan konstanta dari P(x) = 4x³ - 2x² + 7x - 5

2. Hitung nilai P(3) untuk P(x) = x² - 4x + 3

3. Sederhanakan: (2x² + 3x - 1) + (x² - 2x + 4)

4. Kalikan: (x + 3)(x - 2)
