Konsep Pembagian Polinomial

Pembagian polinomial adalah proses membagi suatu polinomial (dividen) dengan polinomial lain (divisor) untuk mendapatkan hasil bagi (quotient) dan sisa (remainder).

Teorema Dasar: Jika P(x) dibagi oleh (x - k), maka:

P(x) = (x - k) × Q(x) + R

Dimana Q(x) adalah hasil bagi dan R adalah sisa pembagian.

Metode Pembagian Polinomial

1. Pembagian Biasa (Long Division)

Metode yang mirip dengan pembagian bilangan, dilakukan langkah demi langkah.

Contoh: Bagi P(x) = 2x³ + 3x² - 5x + 2 dengan (x - 1)

Langkah 1: Bagi suku pertama: 2x³ ÷ x = 2x²

Langkah 2: Kalikan: 2x² × (x - 1) = 2x³ - 2x²

Langkah 3: Kurangkan: (2x³ + 3x²) - (2x³ - 2x²) = 5x²

Langkah 4: Turunkan suku berikutnya: 5x² - 5x

Langkah 5: Bagi: 5x² ÷ x = 5x

Langkah 6: Kalikan: 5x × (x - 1) = 5x² - 5x

Langkah 7: Kurangkan: (5x² - 5x) - (5x² - 5x) = 0

Langkah 8: Turunkan konstanta: 0 + 2 = 2

Langkah 9: Bagi: 2 ÷ x = 0 (sisa)

Hasil: 2x² + 5x + 0 dengan sisa 2

2. Metode Horner

Metode yang lebih efisien untuk pembagian dengan bentuk (x - k).

Langkah-langkah:

  1. Tulis koefisien polinomial berurutan
  2. Tulis nilai k di sebelah kiri
  3. Turunkan koefisien pertama
  4. Kalikan dengan k, tambahkan ke koefisien berikutnya
  5. Ulangi sampai selesai
  6. Angka terakhir adalah sisa

Contoh: Bagi P(x) = 2x³ + 3x² - 5x + 2 dengan (x - 1)

k = 1

Koefisien: 2, 3, -5, 2

2 3 -5 2
k=1 2 5 0
2 5 0 2

Hasil: 2x² + 5x + 0 dengan sisa 2

3. Metode Sintetik (Synthetic Division)

Metode yang mirip dengan Horner, digunakan untuk pembagian dengan bentuk linear.

Teorema Sisa

Jika polinomial P(x) dibagi oleh (x - k), maka sisa pembagiannya adalah P(k).

Contoh: Tentukan sisa pembagian P(x) = x³ - 2x² + 3x - 1 oleh (x - 2)

P(2) = (2)³ - 2(2)² + 3(2) - 1 = 8 - 8 + 6 - 1 = 5

Jadi, sisa pembagian adalah 5

Teorema Faktor

Jika P(k) = 0, maka (x - k) adalah faktor dari P(x).

Contoh: Tunjukkan bahwa (x - 1) adalah faktor dari P(x) = x³ - 3x² + 2x

P(1) = (1)³ - 3(1)² + 2(1) = 1 - 3 + 2 = 0

Karena P(1) = 0, maka (x - 1) adalah faktor dari P(x)

Latihan Soal

1. Bagi P(x) = 3x³ - 2x² + x - 4 dengan (x - 2) menggunakan metode Horner

2. Tentukan sisa pembagian P(x) = x⁴ - 3x² + 2x - 1 oleh (x + 1)

3. Buktikan bahwa (x - 3) adalah faktor dari P(x) = x³ - 6x² + 11x - 6

4. Bagi P(x) = 2x⁴ - 3x³ + x² - 5x + 2 dengan (x² - x - 1) menggunakan pembagian biasa
